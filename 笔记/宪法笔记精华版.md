# 宪法笔记精华版

## 第一部分：宪法概述（基础概念）

### 一、宪法的特征（高频）

1. **根本法**：
   - 内容最根本
   - **法律效力最高**（任何法律不得与宪法抵触）
   - **制定修改最严格**（全人大修宪，常委或1/5代表提议，2/3以上通过）
2. **公民权利的保障书**（核心价值）
3. **民主事实法律化的基本形式**

### 二、宪法的分类（记忆口诀）

| 分类标准             | 类型                           | 例子                           |
| :------------------- | :----------------------------- | :----------------------------- |
| 有无统一法典         | 成文宪法 / 不成文宪法          | 美国（成文） vs 英国（不成文） |
| **制定修改严苛程度** | **刚性宪法 / 柔性宪法**        | 我国是刚性宪法                 |
| 制定机关             | 钦定宪法 / 协定宪法 / 民定宪法 | 清末《钦定宪法大纲》           |

### 三、我国宪法的基本原则

1. 党的领导（最本质特征）
2. 人民主权（一切权力属于人民）
3. 基本人权（国家尊重和保障人权）
4. **权力监督与制约**（注意：我国不是三权分立，是**民主集中制**下的分工制约）
5. 法治原则
6. **民主集中制**（国家机构原则）

## 第二部分：国家性质与形式（国体、政体、结构）

### 一、国体（国家性质）

- **定义**：人民民主专政
- **根本制度**：社会主义制度
- **最本质特征**：中国共产党领导

### 二、政体（政权组织形式）

- **定义**：人民代表大会制度
- **地位**：根本政治制度

### 三、国体与政体的关系（你问过的）

- **国体决定政体，政体反映国体**（内容与形式的关系）

### 四、国家结构形式

- **我国是单一制**（不是联邦制）
- 特征：一部宪法、一个中央政府、公民一个国籍

## 第三部分：基本制度（高频考点密集区）

### 一、选举制度（你问过的）

#### 1. 基本原则

- **普遍性原则**：中国公民 + 年满18 + 未被剥夺政治权利
- **平等原则**：同票同权
- **直接/间接并用**（见下表）
- **秘密投票**：无记名

#### 2. 直接选举 vs 间接选举（你问过的）

| 项目         | 直接选举                                    | 间接选举                   |
| :----------- | :------------------------------------------ | :------------------------- |
| **范围**     | 县、乡两级人大代表                          | 全国、省、设区的市人大代表 |
| **主持**     | 选举委员会                                  | 本级人大常委会             |
| **当选门槛** | 选区全体选民过半数投票 + 参加投票选民过半数 | 全体代表过半数选票         |

#### 3. 差额选举（你问过的）

- **定义**：候选人数 > 应选人数
- **一律差额**，没有等额

### 二、民族区域自治制度（你问过的易错点）

#### 1. 自治地方

- **自治区、自治州、自治县**（注意：民族乡不是！）

#### 2. 自治机关

- **人民代表大会 + 人民政府**（注意：**不包括人大常委会**，这是你问过的A项陷阱！）

#### 3. 任职要求（常考）

- 人大常委会：主任或副主任由自治民族公民担任
- 政府首长：自治区主席、州长、县长**必须**由自治民族公民担任

#### 4. 自治权

- 立法权：制定自治条例、单行条例
- 治安权：经国务院批准，可组织公安部队（唯一有权组织武装的自治地方）

### 三、基本经济制度（土地考点）

#### 1. 所有权归属（你问过的）

| 类型                         | 归属         | 例外                 |
| :--------------------------- | :----------- | :------------------- |
| 矿藏、水流、城市的土地       | **国家所有** | 无例外               |
| 宅基地、自留地、自留山       | **集体所有** | 无例外               |
| 森林、山岭、草原、荒地、滩涂 | **国家所有** | 法律规定属集体的除外 |
| 农村/郊区土地                | **集体所有** | 法律规定属国家的除外 |

#### 2. 土地转让（你问过的）

- **所有权**：不得买卖、非法转让
- **使用权**：可以依法转让

#### 3. 征收征用

- **为了公共利益** + **依法** + **给予补偿**（注意是**补偿**，不是赔偿！）

### 四、特别行政区制度

- **高度自治**：行政管理权、立法权、独立的司法权和终审权
- **中央保留**：外交、防务、任命行政长官
- **备案**：立法会制定的法律须报全国人大常委会备案（**备案不影响生效**）

## 第四部分：公民的基本权利和义务（最爱出题）

### 一、权利分类速记

| 权利类型         | 具体内容                                     | 考点提醒                              |
| :--------------- | :------------------------------------------- | :------------------------------------ |
| **平等权**       | 法律面前平等                                 | 合理差别可以（如少数民族）            |
| **政治权利**     | 选举权与被选举权                             | 年满18 + 未被剥夺政治权利             |
| **政治自由**     | 言论、出版、集会、结社、游行、示威           | **没有罢工**（你问过的）              |
| **宗教信仰**     | 信仰自由                                     | 不得利用宗教妨碍教育、不得强制        |
| **人身自由**     | 不受非法拘禁、逮捕                           | 逮捕需检察院批准/法院决定 + 公安执行  |
| **人格尊严**     | 不受侮辱、诽谤、诬告陷害                     | **禁止用任何方法**（你问过的C项正确） |
| **住宅不受侵犯** | 禁止非法搜查                                 | 区别于“侵犯”                          |
| **通信自由**     | 受法律保护                                   | 检查限于国安+追刑，公安/检察机关      |
| **监督权**       | 批评、建议、申诉、控告、检举                 | “有关机关”查清（不是任何机关）        |
| **获得赔偿权**   | 国家机关侵权可获赔                           | 区别于“补偿”                          |
| **社会经济文化** | 财产权、劳动权、休息权、受教育权、物质帮助权 | **物质帮助权**（你问过的重点）        |

### 二、物质帮助权（你问过的争议题）

- **宪法规定**：公民在**年老、疾病或者丧失劳动能力**的情况下，有权获得物质帮助
- **关键理解**：三者是**或**的关系，符合任一即可
- **“疾病”的理解**：需结合具体法律判断是否需要帮助，但宪法字面未要求“丧失劳动能力”

### 三、被剥夺政治权利（你问过的）

- **剥夺范围**：选举权被选举权 + 六大自由（言论、出版、集会、结社、游行、示威）
- **保留权利**：宗教信仰自由、科学研究、艺术创作、劳动权、受教育权等
- **注意**：出版著作被剥夺，但**创作本身**不剥夺

### 四、公民义务（常考）

- 服兵役是**光荣义务**（不是自由）
- 夫妻双方有**计划生育义务**
- 劳动、受教育既是权利也是义务

## 第五部分：国家机构（最复杂，最易混）

### 一、核心记忆逻辑（你问过的）

#### 1. 任期限制（连续任职不超过两届，共8个）

> **“两委三国三高”**

- **两委**：全国人大常委会委员长、副委员长
- **三国**：国务院总理、副总理、国务委员
- **三高**：国家监察委主任、最高法院院长、最高检检察长

#### 2. 领导体制（你问过的）

| 体制           | 适用机构                               |
| :------------- | :------------------------------------- |
| **集体负责制** | 各级人大及常委会、监察委、法院、检察院 |
| **个人负责制** | 国务院及部委、地方政府、中央军委       |

#### 3. 上下级关系（你问过的核心难点）

> **“监督关系：人大；人常；法院。领导关系：本级人大对本级人常；监察委；检察院；政府。”**

| 关系类型     | 适用                                   | 原因                             |
| :----------- | :------------------------------------- | :------------------------------- |
| **领导关系** | 政府、监察委、检察院、人大对本级人常   | 行政/监察需效率，检察一体化      |
| **监督关系** | 人大对一府一委两院、上级法院对下级法院 | 保证审判独立，权力机关不直接指挥 |

#### 4. 建置区划口诀（你问过的）

> **“省级建置全人大，乡建区划省政府，中间都归国务院。”**

| 层级                               | 审批机关     |
| :--------------------------------- | :----------- |
| 省、自治区、直辖市（设立/撤销）    | **全国人大** |
| 自治州、县、自治县、市（所有变更） | **国务院**   |
| 乡、民族乡、镇（所有变更）         | **省政府**   |

------

### 二、各机构核心职权速记

#### 1. 全国人大（最高权力机关）

- **修宪**（常委或1/5代表提议，2/3通过）
- **制定基本法**（刑、民、国家机构等）
- **选举**：国家主席/副主席、军委主席、监察委主任、最高法院长、最高检检察长
- **决定**：总理（根据主席提名）
- **批准**：省级建置、特别行政区设立
- **战争与和平**

#### 2. 全国人大常委会（常设机关）

- **解释宪法、解释法律**
- **制定非基本法**
- **闭会期间部分补充修改基本法**（不得抵触基本原则）
- **监督一府一委两院**
- **任免**：部长、主任、行长等（根据总理提名）
- **决定**：特赦、动员、紧急状态（全国或个别省）

#### 3. 国家主席（国家元首）

- **根据人大/人常决定**行使职权
- 公布法律、任免、授勋、特赦、动员令、外交、接受使节

#### 4. 国务院（最高行政机关）

- 制定行政法规
- 领导各部委、地方政府
- 编制执行计划预算
- 管理经济、教育、民政、公安、司法行政等
- 批准县级、市级建置和区域划分
- 决定省级范围内部分地区进入紧急状态

#### 5. 监察委（监察机关）

- **性质**：监察机关
- **上下级**：领导关系
- **职权**：监督、调查、处置（职务违法/犯罪）
- **独立行使监察权**，不受行政机关、社会团体、个人干涉

#### 6. 法院（审判机关）

- **性质**：审判机关
- **上下级**：监督关系
- **独立行使审判权**

#### 7. 检察院（法律监督机关）

- **性质**：法律监督机关
- **上下级**：领导关系（检察一体化）
- **独立行使检察权**

## 第六部分：高频易错点总结（你问过的）

### 1. 自治机关 = 人大 + 政府（不包括人大常委会）

### 2. 物质帮助权 = 年老 **或** 疾病 **或** 丧失劳动能力（不是“且”）

### 3. 剥夺政治权利 ≠ 剥夺所有权利（宗教、科研、艺术保留）

### 4. 逮捕 = 检察院批准/决定 + 法院决定 + 公安执行（拘留不一定）

### 5. 补偿 ≠ 赔偿（合法行为给补偿，违法行为给赔偿）

### 6. 备案 ≠ 批准（备案不影响生效）

### 7. 服兵役是义务，不是自由

### 8. 六大自由 = 言论、出版、集会、结社、游行、示威（没有罢工）

### 9. 监督权中“查清事实”的是“有关机关”，不是“任何机关”

### 10. 宅基地、自留地、自留山 = 集体所有（不是国家）

## 第七部分：实战答题技巧

1. **看到“年满十八”+“不分……”** → 优先想**选举权被选举权**（不是批评权）
2. **看到“服兵役”** → 优先想**义务**（不是自由）
3. **看到“罢工”** → 直接排除（宪法没有）
4. **看到“自治机关”** → 检查是否误把“人大常委会”算进去
5. **看到土地题** → 先判断是**所有权**还是**使用权**
6. **看到“拘留”** → 区分是**刑事拘留**（可能需报批）还是**行政拘留**（公安独立决定）
7. **看到上下级关系** → 先想是**行政/监察/检察**（领导）还是**法院**（监督）
8. **看到建置区划** → 直接套三级口诀
